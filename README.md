# E-Commerce SQL Business Analysis

## Project Overview
This project analyzes a Brazilian e-commerce dataset using SQL to uncover insights about platform performance, customer behavior, product demand, and logistics efficiency.

The objective of this project is to demonstrate how SQL can be used to extract meaningful business insights from large transactional datasets and support data-driven decision making.

The analysis answers **22 real-world business questions** covering areas such as revenue generation, customer activity, product performance, and delivery reliability.

---

## Dataset

Dataset used: **Olist Brazilian E-commerce Dataset (Kaggle)**

The dataset contains transactional information from a Brazilian online marketplace and includes multiple relational tables representing customers, orders, products, and payments.

Tables used in this project:

- `customers`
- `orders`
- `order_items`
- `products`
- `payments`

Dataset size:

- ~99,000 orders
- ~96,000 customers
- ~112,000 order items

---

## Database Schema

The dataset consists of multiple relational tables connected through primary and foreign keys.


```
customers
   |
   | customer_id
   v
orders
   |
   | order_id
   v
order_items --------> products
       |                |
       | product_id     | product_id
       v                v
    items           product details

orders --------> payments
   |                 |
   | order_id        | order_id
   v                 v
 order data       payment details
```


This relational schema allows analysis of customer purchases, product sales, and payment transactions across the platform.

## Key Insights

Some of the most important findings from the analysis include:

- Total revenue generated by the platform exceeds **16 million**
- The **average order value** is approximately **160**
- Most customers place only **one order**, indicating low repeat purchase behavior
- The state **SP (São Paulo)** generates the highest revenue and has the most customers
- **Cama, Mesa e Banho (Home Linen)** is the most ordered product category
- The **average delivery time is about 12.5 days**
- Approximately **8% of deliveries are late**
- The state **RR** experiences the **longest delivery times (~29 days)**

---

## Business Recommendations

Based on the analysis, several potential improvements can be suggested:

- Improve logistics infrastructure in regions with long delivery times
- Focus marketing efforts on high-demand product categories
- Implement customer retention strategies to increase repeat purchases
- Optimize shipping operations to reduce delivery delays
- Expand product offerings in high-performing product categories
